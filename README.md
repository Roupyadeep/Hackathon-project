# HostelPulse.io

A personal wellness mirror and community helper for hostel life.

## Features

- **Personal Wellness Dashboard**: Track sleep, steps, classes, and balance score
- **Lifestyle Timeline**: View daily activities and their impact on wellness
- **Gentle Suggestions**: Receive soft, optional wellness prompts
- **Hostel Vibe Map**: See activity levels in different hostel zones
- **Social Synchronization**: Connect with nearby peers for study groups
- **Rewards System**: Earn points for maintaining healthy habits
- **PWA Support**: Installable web app with offline capabilities
- **Dark/Light Mode**: Adaptive theming
- **Mood-Based Themes**: Colors change based on your balance score

## Tech Stack

- React 19
- Tailwind CSS
- React Router
- PWA features

## Getting Started

1. Clone the repository
2. Install dependencies: `npm install`
3. Start the development server: `npm start`
4. Open [http://localhost:3000](http://localhost:3000) in your browser

## User Flow

1. **Landing Page**: Welcome and sign-in prompt
2. **Permissions**: Request access to health data
3. **Dashboard**: Main wellness mirror with balance score
4. **Sync Button**: Update data with loading animation
5. **Vibe Map**: View hostel activity zones
6. **Social Popups**: Occasional connection prompts
7. **Rewards**: Track achievements and streaks

## Design Principles

- Minimal, clean Google-style UI
- Rounded cards with soft shadows
- Smooth animations and transitions
- Responsive for mobile and desktop
- Friendly, non-clinical language
- No pressure, gentle suggestions

## Mock Data

All data is simulated for demonstration purposes. The app uses mock health and activity data to showcase the UI and interactions.
